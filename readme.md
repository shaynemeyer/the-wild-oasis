# The Wild Oasis

A Hotel Managment app written in React with TypeScript.

![The Wild Oasis - All Cabins](docs/images/oasis-screenshot-1.png)

## Tech

### Frontend / UI

- ReactJS
- React Router
- React Hook Form
- Style Components
- UI State - React Context API
- React Query

--- 
### Backend / API

- Supabase - API/Auth/Image Upload
